如果出现无法访问的情况怎么办？

  1. 进入花生壳管理中心
  2. 找到域名诊断
  3. 直到弹出你的公网ip地址为止
  4. 如果ip地址与你的公网IP不一样，说明解析出问题了
  5. 重启你的主机中安装的phdnns,直到它可以正常解析
  6. 如果还不行，关注花生壳官网通知，说不定是花壳免费服务器的解析服务出问题了，就像2016年6月27日这个一样，我找了好久，才发现他官网发能行了，是他们服务器出问题了。。。。。。
