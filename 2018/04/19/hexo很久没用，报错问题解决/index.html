hexo很久没用，报错问题解决

其实这个错是不会影响hexo的使用的，但作为一名精细的技术狗，对这个报错非常不爽，决定解决它。。。。

参考出处：https://github.com/hexojs/hexo/issues/1922

执行：

$ sudo npm uninstall hexo-cli -g
$ sudo npm uninstall dtrace-provider -g
$ sudo npm install hexo-cli -g --no-optional

完美解决。。。。

再次执行: 

hexo server -d

完美。。。。。
