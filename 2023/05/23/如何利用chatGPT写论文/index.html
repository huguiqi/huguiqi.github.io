如何利用chatGPT写论文

对于首次使用 ChatGPT 来写文章、论文的朋友，最开始可能不知道如何「调教」AI，让它产出符合我们要求的内容。

就这个问题，我们可以参考 GitHub 上一个热门的项目 *awesome-chatgpt-prompts**，这个项目列举了众多使用 ChatGPT 时输入的**提示语（prompts）*案例，其中就有一些和写作、写论文相关的：

  • 让 ChatGPT 扮演学者的角色

我希望您扮演一个学者的角色。您将负责研究一个自己选择的主题，并以论文或文章形式呈现研究结果。您的任务是确定可靠的信息来源，以良好结构的方式组织材料，并准确地用引文进行记录。我第一个要求的建议是：“我需要帮助撰写一篇关于面向18-25岁大学生的可再生能源发电现代趋势的文章。”

  • 让 ChatGPT 扮演论文作者

我想让你扮演一名论文作者。你需要研究一个给定的主题，提出一个论点，并创作出一篇既有信息性又具有说服力的文章。我的第一个建议是，“我需要帮助写一篇有关减少环境中塑料垃圾的重要性且有说服力的文章”。

  • 让 ChatGPT 扮演人工智能写作导师

我希望你扮演一个AI写作导师的角色。我会提供一个需要帮助提高写作水平的学生，你的任务是使用自然语言处理等人工智能工具，为学生提供反馈，告诉他们如何改进论文写作。你还应该根据你的修辞知识和写作技巧经验，建议学生更好地表达他们的思想和想法。我的第一个请求是“我需要有人帮我编辑我的硕士论文。”
