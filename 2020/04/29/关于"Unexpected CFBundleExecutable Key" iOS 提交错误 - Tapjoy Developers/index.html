关于”Unexpected CFBundleExecutable Key” iOS 提交错误

9月16日, 2015

“Unexpected CFBundleExecutable Key” 或 “Unexpected
CFBundleSupportedPlatforms value”

我们已经收到一些因为以下错误而导致向苹果商店提交应用失败的报告：

请依照下列步骤去解决这个问题：

在您工程的TapjoyResources.bundle中找到Info.plist，按照下列步骤手动更改

设置下列键值：

CFBundleSupportedPlatforms = iPhoneOS

删除下列键值：

Executable file

如果您用的是Adobe
AIR，您还需解压ANE文件，按照上述步骤更改，然后重新打包ANE文件，具体步骤：

  1. unzip TapjoyExtension.ane -d temp

  2. 在temp/META-INF/ANE/iPhone-ARM/Tapjoy.embeddedframework/Resources/目录下找到TapjoyResources.bundle，然后按照上面步骤修改.

  3. cd temp/META-INF/ANE/

  4. mv iPhone-ARM/platform.xml ios_platform.xml

  5. mv Android-ARM/platform.xml android_platform.xml

  6. /bin/adt -package -target ane ../../../TapjoyExtension.ane extension.xml -swc ../../../TapjoyExtension.swc -platform iPhone-ARM -C iPhone-ARM -platformoptions ios_platform.xml . -platform Android-ARM -C Android-ARM -platformoptions android_platform.xml . -platform default -C default library.swf

如果您还有其他问题，请致信contact
support.
