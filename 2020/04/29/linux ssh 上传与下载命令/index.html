linux ssh 上传与下载命令

上传：

scp
/path/file（这部分为本地的路径）
user（远端目标用户名） host（远端目标IP）:/pathorfile（文件存储路径）

\

下载：

scp
user（远端用户名） host（远端IP）:/path/file（下载文件在远端的路径）
localpathorfile（本地文件存放路径）

\

查看目录或文件：

ssh
user host command ls “/path/*.tgz”


Linux下awk命令合并多文件方法

\

awk ‘{v=FILENAME}!a[v]++1’ /tmp/data/pps/*.ips  >/tmp/all.ips.txt

\

 以上命令是 合并 /tmp/data/pps/目录下所有.ips 结尾的文件 到txt
文件中

\

awk ‘{v=FILENAME}!a[v]++1’
